# Astro とは何か？

Astro は、**「Content-First」** を掲げる現代的な静的サイトゲネレーターです。特にブログやドキュメント、マーケティングサイトなど、文章主体のサイトでその真価を発揮します。

## 圧倒的なスピードの秘密：Zero JS

Astro の最大の特徴は **「アイランド・アーキテクチャ」** です。デフォルトではブラウザに JavaScript を一切送信せず、必要な部分（アイランド）だけに JS を適用します。これにより、初期読み込みが驚異的に速くなります。

### テクニカル・クイックビュー

| 項目 | 内容 |
| :--- | :--- |
| **主要言語** | JavaScript / TypeScript |
| **学習コスト** | 低（HTMLに近い独自構文） |
| **ビルド速度** | 非常に高速 (Viteベース) |
| **得意分野** | ブログ、カタログ、ドキュメント |

## 使い慣れたフレームワークを流用できる

Astro の面白い点は、React, Vue, Svelte などのコンポーネントを、同じプロジェクト内で自在に組み合わせて使えることです。

:::message
既存の資産を活かしつつ、最終的な出力はピュアな HTML に絞る。この「いいとこ取り」が Astro が支持される理由です。
:::

## 結論

今、新しく SSG を始めるなら Astro は最も有力な候補です。このカタログサイトも Astro で構築されており、その性能の高さを証明しています。
