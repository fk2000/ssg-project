# MDX とは何か？

MDX は、Markdown の中で JSX（コンポーネント）を使えるようにした拡張形式です。これにより、単なる文章に強力な「動的要素」を加えることができます。

## なぜ MDX が便利なのか？

1. **再利用性**: 複雑な警告ボックスや図解をコンポーネント化して使い回せる。
2. **インタラクティブ**: Markdown の中にボタンやチャートを埋め込める。
3. **管理の容易さ**: コンピテントの内部ロジックを記事本体から切り離せる。

:::message
この入門書で使用している「メッセージボックス」も、実は MDX の仕組みを使って実装されています。文章を「書く」だけでなく「構成する」楽しみが広がります。
:::
