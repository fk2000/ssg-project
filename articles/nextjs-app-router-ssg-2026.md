# Next.js の SSG

Next.js は、現在最も普及している React フレームワークであり、その SSG (Static Site Generation) 機能は App Router の登場によりさらに強力かつ直感的になりました。

## 特徴

- **ハイブリッド**: ページごとに SSG, SSR を切り替え可能。
- **データ取得**: `fetch` API を使ったシンプルな静的データ取得。
- **エコシステム**: Vercel との親和性が高く、デプロイが非常に容易。

:::message
Next.js は「何でもできる」がゆえに学習コストは高めですが、大規模なプロジェクトや、将来的に動的な機能を追加したい場合には最適な選択肢です。
:::
