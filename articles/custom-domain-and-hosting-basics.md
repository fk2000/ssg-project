# サイトを公開する場所

静的サイトを作ったら、次はどこで公開（ホスティング）するかを決め、独自ドメインを設定しましょう。

## おすすめのホスティング

- **Cloudflare Pages**: 今回も使用。無料枠が大きく、高速。
- **GitHub Pages**: シンプルに始められる。
- **Netlify / Vercel**: プレビュー機能などが非常に強力。

:::message alert
ドメインの DNS 設定が反映されるまでには、最大で 24〜48時間程度かかる場合があります。気長に待ちましょう。
:::
