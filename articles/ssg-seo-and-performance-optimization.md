# 静的サイトだからこそ強い SEO

SSG の最大の武器は、その表示速度です。Google の検索順位にも影響する Core Web Vitals を高水準で達成するためのポイントをまとめます。

## 最適化のポイント

- **メタデータの徹底**: 各ページに個別の `title` と `meta descriptions` を付与。
- **画像の最適化**: WebP 形式の採用や、Lazy Loading（遅延読み込み）の設定。
- **セマンティック HTML**: 適切なタグ (`article`, `nav`, `h1-h6`) の使用。

:::message
Astro をはじめとする最新の SSG は、これらの多くを自動で行ってくれますが、筆者が構造を意識して執筆することが最善の SEO に繋がります。
:::
