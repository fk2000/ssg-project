# 友達を招待しよう

地上絵を完成させたら、誰かに見せたくなりますよね。自前でサーバーを立てるのは大変ですが、**Aternos（アテルノス）** を使えば簡単です。

## Aternos の特徴
- **完全無料**: 広告収入で運営されており、ずっと無料です。
- **ブラウザで完結**: 複雑な設定なしにボタン一つでサーバーを起動できます。
- **Java版対応**: 私たちが必要とする Java 版の Function もしっかりと利用可能です。

## 設定の注意点
サーバーの設定（Options）から、**「コマンドブロック」を ON** にすることを忘れないでください。これをしないと、高度な自動化が動作しません。

:::message
Aternos は手軽ですが、プレイヤーがいないと自動的にスリープモードに入ります。本格的な運用には、後に Cloudflare 等を組み合わせた更なる技術構成もありますが、まずはここでスタートしましょう。
:::
