# 手軽にドット絵を再現

「コマンドを覚えるのはまだ早い...」という方におすすめなのが、ウェブツールを使った方法です。

## 人気のツール
- **Image to Minecraft**: 画像をアップロードするだけで、どのブロックを使えば再現できるか指示してくれます。
- **Mapart Craft**: 地図（マップアイテム）にきれいに映るように計算してブロックを配置する手法です。

## 手動配置の限界
これらのツールで出力された「設計図」を見ながら手でブロックを置いていくのは、気合と時間が必要です。128x128 マスの絵を作るだけでも、数時間はかかってしまいます。
