# プレイヤーから「クリエイター」へ

マインクラフトの楽しみ方は、技術を知るほど広がります。

## ステップ 1：冒険とクラフト
木を切り、家を建て、洞窟を探索する。これが基本の遊び方です。

## ステップ 2：コマンド (Command)
チャット欄に `/tp` や `/give` と打ち込み、世界のルールを操作します。
例：`/weather clear`（天気を晴れにする）

## ステップ 3：ファンクション (Function)
複数のコマンドを一括実行する「プログラム」のようなものです。
外部のテキストファイル（`.mcfunction`）にコマンドを書き込み、ゲーム内で一気に実行します。

:::message
**Function のメリット**:
チャット欄に 1 行ずつ打つのは大変ですが、Function なら数万行のコマンドも 1 回の実行で完了させることができます。これが「巨大な地上絵」を瞬時に描く鍵となります。
:::
