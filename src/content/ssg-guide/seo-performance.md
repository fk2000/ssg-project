# 検索エンジンとユーザーに愛されるサイトへ

SSGを選択した時点で、あなたはすでにパフォーマンスにおいて有利な地点にいます。しかし、さらなる高みを目指すための最適化テクニックが存在します。

## 1. Core Web Vitals の攻略

2026年現在も、LCP（最大視覚コンテンツの表示時間）や CLS（視覚的な安定性）は最重要指標です。

- **画像最適化**: Astro の `getImage` や `Picture` コンポーネントを使い、次世代フォーマット（AVIF/WebP）を自動生成します。
- **フォントの最適化**: `font-display: swap;` の使用や、サブセット化されたフォントのプリロードを実施します。

## 2. メタデータと構造化データ

- **動的なメタタグ生成**: Astro のコンテンツコレクションから `title` や `description` を自動抽出し、全ページに適用します。
- **JSON-LD**: 記事、著者、パンくずリストなどの構造化データを埋め込み、検索結果でのリッチスニペット表示を狙います。

## 3. セマンティック HTML

検索エンジンは「意味のあるタグ」を好みます。`div` を多用せず、`article`, `section`, `nav`, `header` などのセマンティックな要素を正しく使い分けることが、意外にも大きなSEO効果を生みます。

## 結論

パフォーマンスは最高のSEOであり、最高のユーザー体験です。SSGの静的な特性を活かし、一切の無駄を削ぎ落としたサイトを目指しましょう。
