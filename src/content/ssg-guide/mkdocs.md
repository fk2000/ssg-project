# 軽量・強力：MkDocs

MkDocsは、Pythonエコシステムで開発されているSSGで、特に技術ドキュメントの作成において絶大な人気を誇ります。

## なぜ選ばれるのか

- **Material for MkDocs**: 公式テーマ以上の圧倒的なクオリティを持つ「Material」テーマが利用でき、設定だけでモダンなUIが手に入ります。
- **プラグインエコシステム**: 数学記法(MathJax)、図解(Mermaid)、PDF出力など、豊富なプラグインが揃っています。
- **デプロイ設定の簡潔さ**: GitHub Pages や Cloudflare Pages との親和性が非常に高いです。

## 2026年時点の評価
「フロントエンドのコード（JavaScript/Reactなど）を一切書かずに、最高品質の技術ドキュメントを作りたい」というニーズに対して、現在でも最強の選択肢の一つです。
