# 静的サイトジェネレーター (SSG) とは？

静的サイトジェネレーター（Static Site Generator: SSG）は、コンテンツ（Markdownなど）とテンプレートを組み合わせて、事前にHTMLファイルを一括生成するツールです。

## なぜ SSG なのか？

- **爆速**: サーバー側での演算がないため、読み込みが非常に速いです。
- **安全**: データベースがないため、ハッキングの対象となる脆弱性が極めて少ないです。
- **安価**: 静的ファイルのホスティングは、多くのプラットフォームで無料で利用可能です。

このガイドでは、現代的な SSG である Astro を中心に、その魅力と使い方を学んでいきます。

## なぜこのプロジェクトを始めたのか：fujiken-dev の想い

Web開発の世界は日々進化し、新しいツールが次々と登場しています。特に SSG（静的サイトジェネレーター）のエコシステムは非常に豊かですが、同時に初学者にとっては「何を選び、どうプロフェッショナルな環境を構築すればよいのか」が分かりにくい状況でもあります。

私たち **fujiken / fujiken-dev** がこのテーマを掲げ、書籍化を目指して取り組んでいるのには理由があります。それは、一人でも多くの初学者が「ただサイトを作る」レベルを超え、パフォーマンス、SEO、自動化といった**本物のプロフェッショナルへの鍵**を手にしてほしいと願っているからです。

このガイドブックは、単なるツールのカタログではありません。私たちが実際に試行錯誤し、Bitbucket から Cloudflare Pages へのデプロイを成功させたような「生の実践知」を凝縮しています。

一歩ずつ、一緒にプロフェッショナルな Web 開発の道を歩んでいきましょう。
