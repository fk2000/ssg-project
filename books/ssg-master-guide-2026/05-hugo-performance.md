# Hugo の爆速パワー

Hugo は Go 言語で書かれた非常に高速なジェネレーターです。数千ページに及ぶ大規模なサイトでも、1秒足らずでビルドを完了させる圧倒的なパフォーマンスが持ち味です。

## なぜ Hugo なのか？

- **シングルバイナリ**: インポートするだけで即動く。依存関係に悩まされることがありません。
- **テーマの豊富さ**: 多くのハイクオリティなテンプレートが公開されています。
- **柔軟性**: 独自の JSON データなどを使ったサイト構築も得意です。

:::message alert
Hugo 独自のテンプレート言語（Go HTML template）は、React/Vueに慣れた開発者には少し癖が強く感じられるかもしれません。
:::
