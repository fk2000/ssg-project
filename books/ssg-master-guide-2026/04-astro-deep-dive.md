# Astro とは何か？

Astro は、**「Content-First」** を掲げる現代的な静的サイトジェネレーターです。特にブログやドキュメント、マーケティングサイトなど、文章主体のサイトでその真価を発揮します。

## 圧倒的なスピードの秘密：Zero JS

Astro の最大の特徴は **「アイランド・アーキテクチャ」** です。デフォルトではブラウザに JavaScript を一切送信せず、必要な部分（アイランド）だけに JS を適用します。これにより、初期読み込みが驚異的に速くなります。

### テクニカル・クイックビュー

| 項目 | 内容 |
| :--- | :--- |
| **主要言語** | JavaScript / TypeScript |
| **学習コスト** | 低（HTMLに近い独自構文） |
| **ビルド速度** | 非常に高速 (Viteベース) |
| **得意分野** | ブログ、カタログ、ドキュメント |

Astro の面白い点は、React, Vue, Svelte などのコンポーネントを、同じプロジェクト内で自在に組み合わせて使えることです。

## 開発を加速させる：VS Code と Astro による量産体制

Astro を単なるフレームワークとして使うだけでなく、VS Code のエコシステムをフル活用することで、ブログや書籍の執筆効率は劇的に向上します。

### 1. Astro 拡張機能による「型」のサポート
VS Code の Astro 拡張機能をインストールすることで、`.astro` ファイルや Markdown 内のフロントマターに対して強力な補完機能が働きます。記述ミスをエディタが即座に指摘してくれるため、安心して筆を進めることができます。

### 2. Content Collections でスキーマを統一
Astro の「Content Collections」機能を利用し、`src/content/config.ts` で記事の構造を定義しましょう。
これにより、タイトルや日付、タグといったフロントマターに「型」が付き、不整合なデータが混入することを防げます。

### 3. スニペットでフロントマターを瞬時に生成
以下のような VS Code 専用のスニペットを設定しておくことで、`frontmatter` と打ち込むだけで定型文を展開できます。

```jsonc
{
  "Astro Content Frontmatter": {
    "prefix": "frontmatter",
    "body": [
      "---",
      "title: \"${1:タイトル}\"",
      "chapter: ${2:1}",
      "status: \"${3|published,draft|}\"",
      "pubDate: ${CURRENT_YEAR}-${CURRENT_MONTH}-${CURRENT_DATE}",
      "---",
      "$0"
    ]
  }
}
```

:::message
執筆の「儀式」を自動化することで、クリエイティブな「本文」の作成に集中できる環境を構築しましょう。
:::
