# プログラムで絵を描く

ついに本番です。画像解析ツール（Python スクリプトや専用ツール）を使い、画像の 1 ピクセルずつを `/setblock` コマンドに変換したファイルを作成します。

## mcfunction の中身
ファイルの中には、以下のようなコマンドが数千行並びます。
```mcfunction
setblock ~0 ~ ~0 oak_planks
setblock ~1 ~ ~0 stone
setblock ~2 ~ ~0 white_wool
...
```

## 実行方法

1.  作成した `.mcfunction` ファイルを、以下のパスに配置します。
    `datapacks/（任意のフォルダ名）/data/（名前空間）/function/`
    
    :::message alert
    **Java Edition 1.21 での変更点**:
    これまでディレクトリ名は `functions`（複数形）でしたが、**1.21 以降は `function`（単数形）** に変更されました。古い情報を参照していると動作しないため注意が必要です。
    :::

    *   **名前空間パート**: `data/` の直下のフォルダ名が「名前空間」になります。
    *   **実行コマンド**: ゲーム内で実行する際は `/function 名前空間:ファイル名` となります。

2.  ゲーム内で `/reload` を実行して、追加したファイルを読み込ませます。
3.  いよいよ実行です！ `/function 名前空間:ファイル名`

:::message
一瞬のうちに数万個のブロックが設置され、地面に巨大なキャラクターや写真が現れる光景は圧巻です。これこそが Java 版マイクラの真骨頂です。
:::
