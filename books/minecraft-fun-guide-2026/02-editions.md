# Java版と統合版、どちらを選ぶべき？

マインクラフトには大きく分けて 2 つのエディションがあります。

## 1. Java 版 (Java Edition)
PC (Windows, Mac, Linux) で動作する、元祖マインクラフトです。
- **特徴**: MOD（拡張機能）の導入が容易で、コマンドやファンクションの自由度が非常に高い。
- **今回のターゲット**: 地上絵作成や複雑な自動化を行うなら、間違いなく Java 版が最適です。

## 2. 統合版 (Bedrock Edition)
Switch, PS4/5, スマホ, PC で遊べるエディションです。
- **特徴**: 異なるデバイス間でのマルチプレイが容易です。
- **制限**: Java版に比べると、システム内部へのアクセスやカスタマイズに制限があります。

:::message alert
このガイドで紹介する「Function」や詳細な地上絵作成テクニックは、**Java 版**を前提としています。
:::
