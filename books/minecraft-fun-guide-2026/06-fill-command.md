# 一気にブロックを敷き詰めよう

手で置くのは卒業です。`/fill` コマンドを使えば、一瞬で巨大な床を作れます。

## 基本の構文
```mcfunction
/fill <開始座標> <終了座標> <ブロックID>
```

## 例：10x10 の白いキャンバスを作る
自分の足元から前方に 10x10 の白い羊毛を敷き詰める場合：
```mcfunction
/fill ~ ~-1 ~ ~9 ~-1 ~9 white_wool
```

## fill コマンドの限界
`/fill` コマンドは一度に配置できるブロック数が **32,768 個** までと決まっています。また、単一のブロックしか置けないため、複雑な絵を描くのには不向きです。
